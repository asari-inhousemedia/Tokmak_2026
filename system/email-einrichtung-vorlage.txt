   - Hostname: w020e585.kasserver.com
   - Benutzername: info@tokmak-gmbh.de
   - Passwort: Ihr Passwort
6. Server für ausgehende E-Mails:
   - Hostname: w020e585.kasserver.com
   - Benutzername: info@tokmak-gmbh.de
   - Passwort: Ihr Passwort

SIGNATUR EINRICHTEN (iPhone/iPad):
1. Einstellungen > Mail > Signatur
2. "Pro Account" wählen
3. Den Account auswählen
4. Signaturtext eingeben (Klartext, kein HTML möglich auf iOS)


2. APPLE MAIL (Mac)
────────────────────────────────────────

1. Mail > Einstellungen > Accounts > "+" klicken
2. "Anderer Mail-Account" wählen
3. Eingeben:
   - Name: Ihr Name
   - E-Mail-Adresse: info@tokmak-gmbh.de
   - Passwort: Ihr E-Mail-Passwort
4. "Anmelden" klicken
5. Falls manuelle Eingabe nötig:
   - Accounttyp: IMAP
   - Eingangsserver: w020e585.kasserver.com
   - Ausgangsserver: w020e585.kasserver.com

HTML-SIGNATUR EINRICHTEN (Mac):
1. Mail > Einstellungen > Signaturen
2. Account auswählen > "+" klicken > Platzhalter-Text eingeben
3. Mail komplett beenden
4. Finder > Gehe zu > Gehe zum Ordner:
   ~/Library/Mail/V10/MailData/Signatures/
5. Die zuletzt geänderte .mailsignature-Datei öffnen (mit TextEdit)
6. Den HTML-Code nach dem "Mime-Version" Header ersetzen
7. Datei speichern > Im Finder "Datei sperren" (Rechtsklick > Informationen > Geschützt)
8. Mail neu starten


3. MICROSOFT OUTLOOK (Windows / Mac)
────────────────────────────────────────

OUTLOOK FÜR WINDOWS (Neue Version / Microsoft 365):
1. Einstellungen (Zahnrad) > Konten > E-Mail-Konten > Konto hinzufügen
2. E-Mail-Adresse eingeben > "Erweiterte Optionen" > "Ich möchte mein Konto manuell einrichten"
3. IMAP wählen
4. Eingeben:
   - Eingangsserver: w020e585.kasserver.com, Port 993, SSL/TLS
   - Ausgangsserver: w020e585.kasserver.com, Port 465, SSL/TLS
5. Anmelden

OUTLOOK FÜR MAC:
1. Outlook > Einstellungen > Konten > "+" > Neues Konto
2. E-Mail-Adresse eingeben
3. "Nicht IMAP" wählen falls Exchange vorgeschlagen wird
4. IMAP-Daten wie oben eingeben

HTML-SIGNATUR IN OUTLOOK (Windows):
1. Datei > Optionen > E-Mail > Signaturen
2. "Neu" klicken > Name vergeben
3. Im Signatur-Editor: Text eingeben ODER
4. Für HTML: Die Signatur-HTML-Datei herunterladen
5. Im Signatur-Ordner ablegen:
   C:\Users\[Name]\AppData\Roaming\Microsoft\Signatures\
6. Outlook neu starten > Signatur auswählen

HTML-SIGNATUR IN OUTLOOK (Mac):
1. Outlook > Einstellungen > Signaturen
2. "+" klicken > Signatur bearbeiten
3. Hinweis: Outlook Mac unterstützt nur eingeschränkt HTML
   Empfehlung: Signatur als Bild + Text gestalten


4. GMAIL (Webbrowser)
────────────────────────────────────────

KONTO HINZUFÜGEN (E-Mail in Gmail empfangen):
1. Gmail öffnen > Zahnrad > Alle Einstellungen
2. Tab "Konten und Import"
3. "E-Mails von anderen Konten abrufen" > "E-Mail-Konto hinzufügen"
4. E-Mail-Adresse eingeben (z.B. info@tokmak-gmbh.de)
5. IMAP-Daten eingeben:
   - Benutzername: info@tokmak-gmbh.de
   - Passwort: Ihr Passwort
   - POP-Server: Alternativ IMAP verwenden
6. "Auch als Alias senden" konfigurieren:
   - SMTP-Server: w020e585.kasserver.com
   - Port: 465
   - SSL verwenden

HTML-SIGNATUR IN GMAIL:
1. Gmail > Zahnrad > Alle Einstellungen > Allgemein
2. Runterscrollen zu "Signatur"
3. "Neue Signatur erstellen"
4. Für HTML-Signatur:
   a) Die HTML-Signatur-Datei im Browser öffnen
   b) Alles markieren (Strg+A / Cmd+A)
   c) Kopieren (Strg+C / Cmd+C)
   d) In das Signaturfeld in Gmail einfügen (Strg+V / Cmd+V)
5. "Änderungen speichern"

ALLGEMEINE HINWEISE
────────────────────────────────────────

• Benutzername ist immer die vollständige E-Mail-Adresse
• Passwörter finden Sie im Zugangsdaten-Dokument
• Immer SSL/TLS verwenden (nie unverschlüsselt)
• IMAP synchronisiert E-Mails auf allen Geräten
• Bei Problemen: Antivirus/Firewall prüfen (Port 993 und 465 müssen offen sein)
• HTML-Signatur-Datei finden Sie im Google Drive unter "E-Mail-Signatur"
• Für die Webmail-Oberfläche: webmail.all-inkl.com (keine Einrichtung nötig)
